⚡ Dotfiles

Minimal, fast, and aesthetic Linux setup built for a keyboard-driven workflow.

📂 Structure
```
.config/
  btop cava fastfetch fooyin hypr kitty noctalia nvim nwg-look rofi
.local/share/applications/
scripts/
.clang-format
.zshrc
```
🧩 Stack
WM: Hyprland
Terminal: Kitty
Shell: Zsh
Editor: Neovim
Launcher: Rofi
Monitor: btop, fastfetch
Audio: cava, fooyin
🚀 Setup
git clone https://github.com/<your-username>/dotfiles
cd dotfiles
stow .

Install required packages:

hyprland
